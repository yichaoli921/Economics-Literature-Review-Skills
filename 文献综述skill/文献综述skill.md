# 文献综述写作技能

## 技能目标

按照《经济研究》等顶级期刊的学术规范，撰写高质量的文献综述，为实证研究提供理论基础和研究假说。

## 核心理论框架

### 一、文献综述的三个功用（张茂元，2021）

**1. 澄清贡献**
- 明确现有研究的空白点和不足
- 定位本研究的理论贡献和边际贡献
- 避免重复研究，突出创新性

**2. 提供理论框架**
- 从理论视角归类和分析文献
- 构建理论分析框架，为实证研究提供理论基础
- 整合不同理论视角，形成逻辑链条

**3. 提供理论缘由**
- 将自变量和因变量与理论联系起来
- 论证研究的理论意义和学术价值
- 为研究假说推导提供逻辑基础

### 二、文献综述的五个标准

**1. 前沿性**
- 涵盖最新发表的文献（近5-10年）
- 关注前沿研究动态和最新理论发展
- 避免过于陈旧的文献

**2. 覆盖性**
- 涵盖核心文献和相关文献
- 包括不同观点和对立观点
- 覆盖不同领域和跨领域研究

**3. 相关性**
- 所有文献与研究问题紧密相关
- 围绕因变量、自变量及其相互关系展开
- 删除不相关或关联度低的文献

**4. 分析性（而非堆砌）**
- 按照理论视角归类文献，而非简单罗列
- 比较不同观点，批判性评估
- 通过分析提出新问题和新假说

**5. 连贯性**
- 以研究问题为线索贯穿全文
- 上下级标题之间逻辑联系明确
- 形成完整的逻辑链条

### 三、理论创新的要求

**理论创新或新证据**
- 提出新的理论观点
- 提供新的经验证据

**创新理论或新概念**
- 创造性地应用一般理论
- 扩展或整合现有理论
- 提出新概念或新机制

## 结构安排规范

### 一、整体结构

1. **引言**
   - （一）研究问题的提出
   - （二）文献综述的理论功用
   - （三）文献综述的结构安排

2. **主体部分**（按照理论视角组织）
   - 每个理论视角独立成章
   - （一）理论渊源
   - （二）经验研究
   - （三）理论评述与对本研究启示

3. **理论整合与研究假说**
   - （一）理论框架构建
   - （二）理论创新与研究贡献
   - （三）研究假说推导
   - （四）实证设计建议

4. **参考文献**

### 二、标题层级

- 至少使用三级标题（一、（一）和1）
- 下一级标题个数控制在2-6个之间
- 上一级标题和下一级标题之间要写一段话，说明目的、思路和结构

### 三、段落结构

- 一段话里要有层次结构
- 提前构思好要讲几层意思
- 最忌讳一段里面罗列好几项研究，没有概括和分类

## 文献结构图

**要求**：每篇文献综述至少需要有一个文献结构图，放在最前面

**内容**：
- 所综述的内容模块之间的关系
- 能服务和支持什么目的

**示例**（震慑效应文献综述）：

```
震慑效应文献综述结构
│
├── 理论视角
│   ├── 理性威慑理论（A-S模型）
│   ├── 学习效应理论（多期博弈）
│   ├── 信息机制理论（前景理论）
│   └── 空间机制理论（空间经济学）
│
├── 识别方法
│   ├── 双重差分法（DID）
│   ├── 地理距离OLS
│   ├── 模拟实验
│   ├── 随机实地实验
│   ├── 文本分析
│   └── 空间杜宾模型
│
└── 作用机制
    ├── 信息化威慑机制
    ├── 信息机制
    ├── 空间机制
    ├── 学习机制
    └── 示范效应与竞争效应
```

## 写作步骤

### 第一步：明确研究问题

- 确定因变量、自变量及其相互关系
- 明确研究的核心问题和边际贡献
- 围绕研究问题搜集文献

### 第二步：文献搜集与分类

- 搜集核心文献、相关文献、对立观点文献
- 按照理论视角归类文献
- 确保文献的前沿性和覆盖性

### 第三步：构建理论框架

- 从理论视角分析文献
- 整合不同理论视角
- 构建理论分析框架

### 第四步：撰写文献综述

1. **引言**
   - 提出研究问题
   - 说明文献综述的三个功用
   - 概述结构安排

2. **主体部分**
   - 每个理论视角独立成章
   - 每章包括：理论渊源、经验研究、理论评述与启示
   - 强调分析而非堆砌

3. **理论整合与研究假说**
   - 整合理论视角，构建理论框架
   - 明确理论创新和研究贡献
   - 推导研究假说（每个假说说明理论依据）
   - 提出实证设计建议

4. **参考文献**
   - 确保所有引用的文献都有出处
   - 引用恰当，不突兀且不影响行文

### 第五步：检查与修改

按照"八个有"标准检查：
- [ ] 有目标（澄清贡献、提供理论框架、提供理论缘由）
- [ ] 有线索（以研究问题为线索）
- [ ] 有结构（三级标题，每节开头说明目的思路）
- [ ] 有重点（理论逻辑关系和机理）
- [ ] 有关联（所有内容与研究问题紧密相关）
- [ ] 有分析（批判性评估，提出新问题和新假说）
- [ ] 有图表（至少一个文献结构图）
- [ ] 有出处（参考文献完整）

## 关键要点

### 1. 避免堆砌，强调分析

**错误做法**：
- 简单罗列文献：A说了什么，B说了什么，C说了什么
- 没有分类和概括
- 没有比较和批判

**正确做法**：
- 按照理论视角归类文献
- 比较不同观点，批判性评估
- 通过分析提出新问题和新假说

### 2. 突出理论逻辑

**重点**：
- 由因变量和自变量组成的理论逻辑关系、机理或机制
- 观点是主要的，角度和方法是辅助性的
- 区分主次

### 3. 确保连贯性

**方法**：
- 以研究问题为线索贯穿全文
- 上下级标题之间逻辑联系明确
- 形成闭环思维（从问题起源到假说提出）

### 4. 研究假说要有理论依据

**要求**：
- 每个研究假说都要说明理论依据
- 假说推导要逻辑严密
- 假说要可检验

## 示例：震慑效应文献综述

### 研究问题
地铁开通是否对企业税收遵从产生震慑效应？这一效应通过何种机制发挥作用？

### 理论框架
- 理性威慑理论：查处概率和处罚率影响预期成本
- 学习效应理论：信息传递效率和学习效应影响震慑持续性
- 信息机制理论：信息完备性和信息干预有效性影响遵从度
- 空间机制理论：空间距离和空间关联影响震慑效应

### 研究假说
- 假说1（主效应）：地铁开通对企业税收遵从产生正向震慑效应
- 假说2（空间机制）：通过空间可达性发挥作用
- 假说3（集聚机制）：通过产业集聚发挥作用
- 假说4（异质性）：对不同类型企业效应不同
- 假说5（持续性）：具有持续性但随时间衰减

### 理论创新
- 拓展震慑效应的研究视角（空间基础设施变革）
- 整合四个理论视角
- 提出空间震慑效应的新机制

## 注意事项

1. **不要跑题**：始终围绕研究问题展开，删除不相关内容
2. **不要堆砌**：按照理论视角归类，强调分析而非罗列
3. **不要忽视理论**：文献综述的核心是理论逻辑关系，而非方法细节
4. **不要缺少结构图**：至少需要一个文献结构图
5. **不要缺少出处**：所有引用都要有参考文献
6. **不要忘记假说**：文献综述最终要提出新问题和新假说

## 适用场景

- 期刊论文文献综述
- 学位论文文献综述
- 开题报告文献综述
- 工作论文文献综述
