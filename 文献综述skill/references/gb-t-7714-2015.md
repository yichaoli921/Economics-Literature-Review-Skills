# GB/T 7714-2015 参考文献著录规则

本文档详细说明 GB/T 7714-2015 标准的引用格式规范，以及与 BibTeX 的对应关系。

---

## 基本格式

### 期刊论文

**中文**：
```
[序号] 主要责任者. 题名[文献类型标识]. 刊名, 年, 卷(期): 起止页码. DOI.
```

**英文**：
```
[序号] Author A, Author B. Title[文献类型标识]. Journal Name, Year, Vol(Issue): Pages. DOI.
```

**示例**：
```
[1] 张三, 李四, 王五. 基于深度学习的医学图像分析研究进展[J]. 中华医学杂志, 2023, 103(15): 1156-1160. DOI:10.3760/cma.j.cn112137-20230101-00001.

[2] LeCun Y, Bengio Y, Hinton G. Deep learning[J]. Nature, 2015, 521(7553): 436-444. DOI:10.1038/nature14539.
```

### 专著（书籍）

**中文**：
```
[序号] 主要责任者. 书名[文献类型标识]. 版本项. 出版地: 出版者, 出版年: 引文页码.
```

**英文**：
```
[序号] Author. Book Title[文献类型标识]. Edition. Place: Publisher, Year: Pages.
```

**示例**：
```
[3] 周志华. 机器学习[M]. 北京: 清华大学出版社, 2016: 125-130.

[4] Goodfellow I, Bengio Y, Courville A. Deep Learning[M]. Cambridge: MIT Press, 2016.
```

### 会议论文

**中文**：
```
[序号] 主要责任者. 题名[文献类型标识]//编者. 会议文集名. 出版地: 出版者, 出版年: 起止页码.
```

**英文**：
```
[序号] Author. Title[文献类型标识]//Editor. Proceedings Name. Place: Publisher, Year: Pages.
```

**示例**：
```
[5] 王小红, 赵大明. 卷积神经网络优化算法研究[C]//中国计算机学会. 第30届中国计算机大会论文集. 北京: 电子工业出版社, 2023: 45-52.

[6] He K, Zhang X, Ren S, et al. Deep residual learning for image recognition[C]//Proceedings of the IEEE Conference on Computer Vision and Pattern Recognition. Piscataway: IEEE, 2016: 770-778.
```

### 学位论文

**中文**：
```
[序号] 主要责任者. 题名[文献类型标识]. 保存地: 保存单位, 年份.
```

**英文**：
```
[序号] Author. Title[文献类型标识]. Place: Institution, Year.
```

**示例**：
```
[7] 刘晓明. 基于Transformer的中文命名实体识别研究[D]. 北京: 北京大学, 2023.

[8] Sutskever I. Training recurrent neural networks[D]. Toronto: University of Toronto, 2013.
```

### 电子资源

**中文**：
```
[序号] 主要责任者. 题名[文献类型标识/文献载体标识]. (更新或修改日期)[引用日期]. 获取和访问路径. DOI.
```

**英文**：
```
[序号] Author. Title[EB/OL]. (Date)[Access Date]. URL. DOI.
```

**示例**：
```
[9] 中国互联网络信息中心. 第52次中国互联网络发展状况统计报告[R/OL]. (2023-08-28)[2024-01-15]. https://www.cnnic.net.cn/

[10] Vaswani A, Shazeer N, Parmar N, et al. Attention is all you need[EB/OL]. (2017-06-12)[2024-01-15]. https://arxiv.org/abs/1706.03762.
```

## 作者著录规则

### 中文作者

- **3 人以内**：全部列出，用逗号分隔
- **超过 3 人**：列出前 3 人，加 "等"

**示例**：
```
[1] 张三, 李四, 王五. ...                    # 3人
[2] 张三, 李四, 王五, 等. ...                # 超过3人
```

### 英文作者

- **姓在前，名缩写在后**（如 Smith J）
- **3 人以内**：全部列出，用逗号分隔
- **超过 3 人**：列出前 3 人，加 "et al."

**示例**：
```
[1] Smith J, Johnson A, Williams B. ...      # 3人
[2] Smith J, Johnson A, Williams B, et al. ...  # 超过3人
```

## 编号规则

### Literature Survey 专用编号

- **中文文献**：编号以 `C` 开头（C1, C2, C3...）
- **英文文献**：编号以 `E` 开头（E1, E2, E3...）

### 通用编号

按文中出现顺序：
- 混合编号：`[1], [2], [3]...`

## 文中引用格式

### 单篇引用

| 类型 | 格式 | 示例 |
|------|------|------|
| 中文 | `[C1]` | 张三等[C1]研究发现... |
| 英文 | `[E1]` | Smith et al.[E1] proposed... |

### 多篇连续引用

| 类型 | 格式 | 示例 |
|------|------|------|
| 中文 | `[C1-C3]` | 早期研究[C1-C3]表明... |
| 英文 | `[E1-E3]` | Previous work[E1-E3] showed... |

### 多篇不连续引用

| 类型 | 格式 | 示例 |
|------|------|------|
| 中文 | `[C1,C3,C5]` | 相关研究[C1,C3,C5]显示... |
| 英文 | `[E1,E3]` | Studies[E1,E3] demonstrated... |

### 作者提及式引用

**中文**：
```
张三等[C1]通过实验发现...
李四等[C2-C4]进一步指出...
```

**英文**：
```
Smith et al.[E1] proposed a novel approach...
Johnson and colleagues[E2] demonstrated that...
Recent studies[E3,E5] have shown...
```

## BibTeX 字段映射

### 期刊论文 (Article)

| GB/T 7714 | BibTeX 字段 | 示例 |
|-----------|-------------|------|
| 主要责任者 | `author` | `author = {LeCun, Yann and Bengio, Yoshua}` |
| 题名 | `title` | `title = {Deep learning}` |
| 刊名 | `journal` | `journal = {Nature}` |
| 年 | `year` | `year = {2015}` |
| 卷 | `volume` | `volume = {521}` |
| 期 | `number` | `number = {7553}` |
| 页码 | `pages` | `pages = {436--444}` |
| DOI | `doi` | `doi = {10.1038/nature14539}` |

**BibTeX 示例**：
```bibtex
@article{LeCun2015Deep,
  author = {LeCun, Yann and Bengio, Yoshua and Hinton, Geoffrey},
  title = {Deep learning},
  journal = {Nature},
  year = {2015},
  volume = {521},
  number = {7553},
  pages = {436--444},
  doi = {10.1038/nature14539},
  abstract = {Deep learning allows...}
}
```

| GB/T 7714 字段 | BibTeX 字段 | 必填 | 说明 |
|---------------|-------------|------|------|

## BibTeX 输入与 GB/T 7714 输出示例

**输入 (BibTeX)**:
```bibtex
@article{Smith2023Novel,
  author = {Smith, John and Johnson, Alice and Williams, Bob},
  title = {A Novel Approach to Medical Image Analysis},

  year = {2023},
  volume = {42},
  number = {3},
```

## 常见问题

### Q: 如何处理没有卷号/期号的期刊文章？

A: 可以省略卷号和期号：
```
[1] 作者. 题名[J]. 刊名, 年: 页码.
```

### Q: 如何引用在线发表但无页码的文章？

A: 使用 DOI 或文章编号：
```
[1] 作者. 题名[J]. 刊名, 年. DOI:xxxxx.
```

### Q: 中文文献在 BibTeX 中如何处理？

A: 保持中文原文，设置 language 字段：
```bibtex
@article{张三2023深度,
  author = {张三 and 李四},
  title = {基于深度学习的图像识别研究},
  journal = {计算机学报},
  year = {2023},
  language = {zh}
}
```

### Q: arXiv 预印本如何引用？

A: 优先查找发表版本，如只有预印本：
```
[1] 作者. 题名[EB/OL]. arXiv预印本, 年. arXiv:xxxx.xxxxx.
```

或 BibTeX:
```bibtex
@misc{作者年标题,
  author = {作者},
  title = {标题},
  year = {年},
  note = {arXiv preprint arXiv:xxxx.xxxxx}
}
```
